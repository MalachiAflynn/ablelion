See the roadmap Current build

What this is

ableLion is one continuous line of homemade drones, each one a little more capable than the last, all running flight code written from the ground up on an ESP32 instead of off-the-shelf firmware.

The point is not to ship a product. The point is to learn out loud, keep every lesson and every failure on the record, and let the work compound over years.

Why public

Sell one thing and it is a company. Until then it is a build log, and a build log is the most honest version of a portfolio: it shows the process, not just the polished result.

Everything here is written as it happens. The blocked builds, the wrong parts, the fixes that finally worked. That trail is the whole point.
