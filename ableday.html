ableDay

A personal organization hub for my Mac that brings my Google Calendar and my daily task list into one local dashboard. It connects to the Google Calendar API (read & write) and keeps my tasks in a local database, so the whole picture of my day lives in one place — and, like the rest of ableLion, it runs entirely on my machine.

STATUS: BUILT & RUNNING LOCALLY · GOOGLE CALENDAR + LOCAL TASKS
Open dashboard (local) Requires the app running locally on this machine.

What it is

A single screen for the day: a Today view that puts my calendar agenda next to my task list, and a Week view that lays out the next seven days. Calendar events sync from Google; tasks are mine to add, check off, and prioritise.

The scope stays narrow on purpose: single-user, no cloud, no login. One person, one machine, one place to see what today actually looks like.

Why build it

I was living between Google Calendar and a separate to-do app and never seeing both at once. I wanted one hub that shows my real schedule and my tasks together — on my terms, in the ableLion style.

Same principle as the rest of the family: own the dashboard, keep the data and the OAuth tokens on the machine. Nothing about my day leaves it.

What's inside

A tabbed dashboard backed by the Google Calendar API and a local task store. Calendar sync is on-demand and incremental; tasks are instant local writes.

The stack

Deliberately the same shape as ableBody and ableBudget — swap the data source for the Google Calendar API and the privacy model carries straight over.
