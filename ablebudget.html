AbleBudget

a personal, local budgeting dashboard

AbleBudget is a local-first budgeting dashboard that links read-only to your bank via Plaid. It is a single-user app: no cloud, no logins, and the database stays on your machine. This page tracks progress and documentation as the app is built.

What it is

A desktop-forward web app to link your accounts (via Plaid), import transactions into a local SQLite DB, set monthly budgets by category, and view budget vs actual charts.

Privacy-first: access tokens and transaction data never leave your machine.

Where it stands

Active development. Backend + frontend scaffolded; Plaid integration and sync in progress. See the repo for build & run instructions.
